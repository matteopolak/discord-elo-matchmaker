## Features

* Multi-guild support
* Parties
* Factions
* Custom modes
* Support for multiple games

## Installation

```powershell
git clone "git@github.com/matteopolak/matchmaker.git"
cd matchmaker
yarn install
```

## Deploy

```powershell
yarn run start
```
